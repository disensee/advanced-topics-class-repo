What is this?

'this' can change depending on the context!

  1. For functions (not methods), 'this' refers to the window object.
  2. For methods, 'this' refers to the object through which you call the method.
  3. For event handlers, 'this' refers to the object that triggered the event.

The new 'class syntax' in JavaScript clears up a lot of this confusion and acts more like 'this' in Java
